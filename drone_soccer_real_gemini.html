鍵盤控制說明

  • W / ↑ : 前進 (Pitch Forward)
  • S / ↓ : 後退 (Pitch Backward)
  • A / ← : 向左 (Roll Left)
  • D / → : 向右 (Roll Right)
  • Q / E : 逆/順時針旋轉 (Yaw)
  • Space / Alt : 長按連續 升高 / 降低
  • C : 切換鏡頭視角 (跟隨 / 座艙 / 自由)
  • T : 一鍵起飛 (平穩上升至 1.5 米並定高)
  • L : 一鍵降落 (平穩降落至中心並熄火)
  • R : 重置無人機 (回到場地中央)
  • M : 靜音切換

對戰規則

  • 球門光環與您的無人機 同色 = 自家球門（不計分，需防守）
  • 球門光環與您的無人機 異色 = 對手球門（穿越才計分）
  • 您的機頭預設面向對手球門方向
  • 電腦控制的守門無人機會鎮守對手球門，撞擊會使您彈開
